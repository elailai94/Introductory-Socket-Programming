# Introductory Socket Programming
### About
Jotto is written entirely in Java and builts on most UNIX-like operating systems (i.e.: Linux or Mac) and Windows. 

### Compilation
```Bash
make
```

### Clean Build
```Bash
make clean
```

### Execution
```Bash
./server.sh <req_code>
./client.sh
```
